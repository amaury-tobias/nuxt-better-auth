::note
**Prerequisites:** A Nuxt 3.10+ project. If you don't have one, run `npx nuxi init my-app`.
::

This guide gets you to a working email/password login using NuxtHub (SQLite) for persistence.
For detailed explanations, see [Installation](/getting-started/installation).

::steps

### Install

::code-group{sync="pm"}
```bash [pnpm]
pnpm add @onmax/nuxt-better-auth better-auth @nuxthub/core
```
```bash [npm]
npm install @onmax/nuxt-better-auth better-auth @nuxthub/core
```
```bash [yarn]
yarn add @onmax/nuxt-better-auth better-auth @nuxthub/core
```
::

### Configure Nuxt

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxthub/core', '@onmax/nuxt-better-auth'],
  hub: {
    db: 'sqlite' // Use NuxtHub's SQLite for local development
  },
})
```

### Set Environment Variables

```ini [.env]
NUXT_BETTER_AUTH_SECRET="generate-a-32+ char secret"
NUXT_PUBLIC_SITE_URL="http://localhost:3000"
```

### Create the Server Config

```ts [auth.config.ts]
import { defineServerAuth } from '@onmax/nuxt-better-auth'

export default defineServerAuth(() => ({
  emailAndPassword: { enabled: true },
}))
```

### Create the Client Config

```ts [client-auth.config.ts]
import { createAuthClient } from 'better-auth/vue'

export function createAppAuthClient(baseURL: string) {
// Module injects baseURL at runtime
  return createAuthClient({ baseURL })
}
```

### Add a Login Page

```vue [pages/login.vue]
<script setup lang="ts">
definePageMeta({ auth: 'guest' })

const { signIn } = useUserSession()
const email = ref('')
const password = ref('')

async function login() {
  await signIn.email({
    email: email.value,
    password: password.value,
    onSuccess: () => navigateTo('/dashboard'),
  })
}
</script>

<template>
  <form @submit.prevent="login">
    <input v-model="email" type="email" placeholder="Email" />
    <input v-model="password" type="password" placeholder="Password" />
    <button type="submit">
      Login
    </button>
  </form>
</template>
```

### Add a Protected Page

```vue [pages/dashboard.vue]
<script setup lang="ts">
definePageMeta({ auth: 'user' })
const { user, signOut } = useUserSession()
</script>

<template>
  <h1>Welcome {{ user?.name }}</h1>
  <button @click="signOut()">
    Logout
  </button>
</template>
```

### Run the Dev Server

::code-group{sync="pm"}
```bash [pnpm]
pnpm dev
```
```bash [npm]
npm run dev
```
```bash [yarn]
yarn dev
```
::

::

## What Just Happened?

You now have:
- A SQLite database with user and session tables (auto-migrated)
- API routes at `/api/auth/*` handling registration, login, and logout
- A reactive `useUserSession()` composable available throughout your app
- Route protection ready to use via `routeRules` or `definePageMeta`
